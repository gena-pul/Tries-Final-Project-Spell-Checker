# Trie Spell Checker (C++)

This project implements a simple **Trie (Prefix Tree)** in C++ for fast word storage and lookup. It supports inserting, searching, deleting words, and basic spell-checking with suggestions.

## Features
- Insert words into the Trie
- Search for existing words
- Delete words safely using recursion
- Spell check with prefix-based suggestions
- Menu driven console interface

## How to Build & Run

### 1. Compile
```bash
g++ main.cpp -o trie
./trie
